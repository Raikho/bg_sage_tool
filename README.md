
Can use this AutoHotKey tool to input bulk data into a program like Sage, where normally each entry has to be entered manually one at a time.

---

**Features:**
* Captures clipboard content whenever clipboard changes (or pressing "Read Clipboard button")
  
* Shows the captured content in the ui (ignores spaces, dollar signs, commas)
  
* Upon pressing **F5**, the Clipboard will be pasted one field at a time, with "TAB" inbetween each field
  * Upon pressing **F8**, it will paste with the "Down Arrow" key pressed inbetween each field
